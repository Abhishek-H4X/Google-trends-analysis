# 📊 Google Trends Analysis using Python

A professional data analysis project that explores **Google Trends** data using **Pytrends**, **Pandas**, and **Matplotlib**. This project compares keyword popularity over time and across different regions through data visualization and exploratory analysis.

---

## 🚀 Project Overview

This project demonstrates how to collect, analyze, and visualize Google Trends data using Python. It provides insights into search interest by comparing keywords such as **ChatGPT** and **AI**, helping identify search patterns and regional popularity.

---

## ✨ Features

* 📈 Compare multiple Google Trends keywords
* 🌍 Analyze search interest by region
* 📅 Visualize interest over time
* 📊 Generate bar charts and line graphs
* 🧹 Perform data sorting, filtering, and analysis using Pandas
* 📁 Export analyzed data for further use

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/google-trends-analysis.git
```

Navigate to the project folder:

```bash
cd google-trends-analysis
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook:

```bash
jupyter notebook
```
