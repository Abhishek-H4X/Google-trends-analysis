# Google-trends-analysis
Project Title
Google Trends Analysis using Python
Project Description
Analyzes Google Trends data for keywords such as ChatGPT and AI using Pytrends.
Features
Compare multiple keywords
Interest over time
Interest by region
Data visualization
Export results
Technologies
Python
Pandas
Matplotlib
Pytrends
Jupyter Notebook

How to Run

pip install pytrends pandas matplotlib

Then open the notebook and run all cells.

Add screenshots
